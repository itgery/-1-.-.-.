Проект представаляет из себя консольное приложение, которое реализует управление коллекцией объектов в интерактивном режиме.
Для хранения обьектов класса "Person" используется коллекция типа java.util.HashSet.
Программа поддерживает все необходимые команды и позволяет создавать "профили" множества людей, включая в них выбранные критерии(такие как имя, рост, день рождения и т.д.)

В этом проекте используется система сборки maven.

Требования к запуску: наличие JDK, Maven(для сборки) и Git(опционально).

Команды сборки:
mvn package # (для сборки "с нуля")
mvn clean package # (для сборки с очисткой даанных предыдущей сборки)

Команды запуска:

Windows Power Shell:

$env:PERSON_DATA_FILE="data.xml"; java -jar target/Lab1_2sem.jar # (можно ввести в 2 строки, исключая знак ; и делая на его месте разрыв строк)

Windows (CMD): 

set PERSON_DATA_FILE=data.xml && java -jar target\Lab1_2sem.jar (можно ввести в 2 строки, заменяя && на разрыв строк)

Linux/Mac:

export PERSON_DATA_FILE=data.xml

java -jar target/Lab1_2sem.jar
